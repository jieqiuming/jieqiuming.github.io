一、如何管理组织模板

  1. SCRIPT TAGS：页面内嵌套，寄存在script标签元素中，根据DOM操作获取模板内容
  2. 页面外组织模板文件，通过jquery.load函数加载模板内容，并且注入到DOM元素内。本质上是利用xhr对象进行ajax调用： 
var xhr = new XMLHttpRequest(); 
xhr.onreadystatechange = function () {
    if (xhr.readyState === 4) {     
        //do something with xhr.responseText
    }   
};      
xhr.open('GET', '/template.html');
xhr.send();
